# Netlify 部署指南

## 最简单的方式：Netlify Drop

### 步骤（2分钟）：

1. **打开 Netlify Drop**
   👉 https://app.netlify.com/drop

2. **拖拽文件夹**
   - 打开文件夹：`C:\Users\11346\.openclaw\workspace\projects\copytrading\frontend\dist`
   - 把 `dist` 文件夹里的所有文件**拖拽**到网页上
   - 或者拖拽 `deploy.zip` 文件

3. **完成！**
   - 自动生成链接，如：`https://copytrading-xxxxx.netlify.app`
   - 复制链接即可分享

---

## 其他方式

### GitHub Pages（免费）
1. 创建 GitHub 仓库
2. 上传 dist 文件夹内容
3. 开启 GitHub Pages

### Cloudflare Pages（免费）
1. 注册 Cloudflare
2. 上传 dist 文件夹
3. 获得全球 CDN 加速链接
